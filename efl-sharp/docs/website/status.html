For now this is just a basic page to get the site up.

The project is currently active in SVN.

To download use the command:
svn co svn://svn.myrealbox.com/source/trunk/efl-sharp
or get a tarball from here. The basic plan thus far is this:
  1. Complete Evas# Binding
  2. Complete Enough of Ecore_Evas# and Ecore# to allow program execution
  3. Start Edje# while creating an application to draw Edje interfaces


The application planned will be the test bed for most libs.

The interface will be created in Edje, and will use Edje# to hook the events.
Ewl# will be used for an dialogs created for the application.
Evas# will be used to manipulate and draw the objects on the screen. Ecore# will be used for events, window creation, and pretty much everything else :)
