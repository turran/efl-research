Welcome to the home of EFL-Sharp
The Mono bindings for the Enlightenment Foundation Libraries.


For those of you who don't know, Enlightenment is a desktop shell created for the X Windows System for Unix and platforms such as Linux, Solaris, and BSD systems.

The EFL are libraries that were created to support the creation of the Enlightenment desktop environment. The EFL-Sharp libraries will allow a developer to create applications using Enlightenment's technologies within the Mono platform using the C# language.
